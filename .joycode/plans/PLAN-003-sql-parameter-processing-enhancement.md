# SQL 参数处理增强计划

## 任务概述
改进 VectorStoreTaskDecomposer 类中的 SQL 参数处理功能，特别是针对 IN 子句参数的处理。当前实现简单地将提取的参数值直接替换到 SQL 模板中，没有考虑参数类型和格式。本计划旨在增强参数提取和处理能力，确保生成的 SQL 语句符合 SQL 规范和 Java 规范，提高系统的健壮性和安全性。
